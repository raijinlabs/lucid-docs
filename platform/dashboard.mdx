# Dashboard

The [Lucid Dashboard](https://app.lucid.foundation) provides a visual interface for managing your AI infrastructure.

## Sections

### Overview

The landing page shows:
- Total requests, tokens, and costs this billing period
- Active passports and their health status
- Recent receipts with verification status
- Epoch anchoring progress

### Passports

Browse and manage all registered passports:
- Filter by type (Model, Agent, Compute, Tool, Dataset)
- View passport metadata and capabilities
- Create, edit, or deactivate passports
- Sync from HuggingFace or other sources

### Receipts

Search and verify inference receipts:
- Full-text search by receipt ID, model, or content hash
- Filter by date range, model, or verification status
- Verify receipt signatures inline
- Export receipts as CSV or JSON

### Epochs

Monitor on-chain anchoring:
- View epoch history with receipt counts
- Check Solana transaction status
- Verify MMR roots against on-chain data
- See pending receipts awaiting next epoch

### Agents

Manage AI agent configurations:
- Create and configure multi-channel agents
- View agent execution history and audit trails
- Monitor session budgets and tool usage
- Configure channel integrations (Telegram, Slack, Discord, WhatsApp)

### Settings

Configure your workspace:
- **API Keys** — Create, rotate, and revoke API keys
- **Billing** — Plan management and invoices
- **Team** — Invite members and manage roles
- **Connections** — OAuth integrations for MCPGate
- **Webhooks** — Configure event notifications
- **Usage** — Detailed metering and cost breakdown

## Quick Actions
